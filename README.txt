Implementation of the stable marriage solution

Takes input from stdin eg:

3 		//number of each node to be matched
2 1 3	//1st blue node preferences
3 2 1
1 3 2
2 1 3	//1st pink node prefs
3 2 1
1 3 2
2		//second set of nodes to be matched
1 2
1 2
2 1
2 1
0		//termination input

Output on above input:

Instance 1:
2		//1st blue is paired with 2nd pink
3		//2nd blue is paired with 3rd pink
1	
Instance 2:
2
1
